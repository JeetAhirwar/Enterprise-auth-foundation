# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, respectful, and inclusive environment for everyone.

## Expected Behavior

- Be respectful.
- Be constructive.
- Accept feedback professionally.
- Help others learn.
- Use welcoming language.

## Unacceptable Behavior

- Harassment
- Personal attacks
- Hate speech
- Discrimination
- Trolling
- Spamming

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct and may remove comments, commits, issues, or pull requests that violate these guidelines.

Repeated violations may result in temporary or permanent bans from the project.

## Contact

For concerns regarding the Code of Conduct, open a GitHub issue or contact the maintainers.
